The flow of the internet:
"You" make a request to the internet that then sends an HTTP request to the server (which are pieces of hardware sitting in closets, storing files), the server then sends back the relevant info to the internet who then displays it to you!

HTML and what it is made of

HTML: (Hypertext Markup Language), basis for everything on the web
HTML is made up of text content ("what you see") and markup which dictates "what it looks like".
Text is normal text and markup is denoted by tags < >.

HTML Tags and Elements
HTML markup is made up of elements and tags. The element specifies what we want the computer to do that is "special" while the tags denote the element.
There are inline elements and there are block elements. Block elements create an invisible box of sorts around their content-- think of it like a form, etc. For any further information, check out this website: Udacity.com and head to the Intro to Programming Nanodegree. :]
